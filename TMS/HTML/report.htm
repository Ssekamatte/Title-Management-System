The Reports essentially provides the user with the capability to view, export the Calender of Activities to PDF, Word, Excel and Comma Separated Values (CSV). Reports for the calender is accessed as shown in the screen below.

 

 

We have a search grid parametrized with: Organ / Institution, Planning Period (Months), Directorate, Planning Period (Year), Department / Division, Venue, Nature Of Activity, and Approved / Draft Activities all of which are drop-downs.

 

 

 

 

Click the search button, which loads the report after filling in the desired search parameters by a user
