Key Features

Some important features of the Data Grid View are:

1.Responsive - Provides three level of responsive modes which includes redesigned UI for Mobile.

2.Sorting and grouping - Supports n levels of sorting and grouping.

3.Filtering - Offers Excel-like filtering to filter data.

4.Editing - Offers two editing modes for inserting, editing, and deleting records in a grid.

5.Paging - Provides the option to easily switch between pages using the pager bar.

6.Reordering - Allows you to drag any column and drop it at any position in the Grid’s column header row, allowing columns to be repositioned at the required place.

7.Resize columns - Grid provides option for resizing the columns.

8. - Offers options for specifying summary rows and columns.

9.Detail template - Offers to render the detail row for the corresponding expanded master row.

10.Columns - Offers the option to specify unbound columns.

11.Free text Search - This searches through the grid

7). Selection

The selection provides an interactive support to highlight the row, cell or column that you select. Selection can be done through simple mouse down or keyboard interaction. Row selection is enabled by pressing  “Ctrl + mouse left” click and for continuous row selection press “Shift + mouse left” click on the grid rows. To unselect the selected rows, press “Ctrl + mouse left” click on the selected row.
