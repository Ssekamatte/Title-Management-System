Scheduler is an event calendar that manages the list of various activities (events/appointments) in different available views (day, week, workweek, month and agenda) for various resources. It is mainly for tracking the user appointments and allows them to create a new or edit/delete the older ones.

 

 

 

Views

The number of days and its associated appointments are usually grouped together in calender Scheduler to organize different views. The available view options in Scheduler are as follows,

 

1.Day

         

 

2.Week

         It’s a view displaying a count of 7 days (from Sunday to Saturday) with all its related appointments. The first day of the week can also be changed. The default value of this FirstDayOfWeek depends on the current                culture (language) used in the Scheduler.

 

 

3.Work-week

         Work week view displays the working days of the week (count of 5 days) and its associated appointments. It is also possible to customize the days to be displayed in the work week view. By default, it renders from                Monday to Friday (5 days).

 

 

4.Month

         Month view displays the entire days of a particular month and all its related appointments. An alternative way to navigate to a particular date in a day view directly from Month view, clicking on the appropriate month                cell date header will do so. If the week date range column is clicked, it will navigate to the corresponding week view.

 

 

         

 

5.Agenda

This View option lists out the appointments in a grid-like view for the next 7 days by default from the current date. we can also customise count of the number of appointments to be listed in this view can be customized

 

 

 

 

 

 

Usually these view options are displayed as a toolbar in the date-header section of the Schedule control.  By default, the calender's active view is Month view.

 

Resources

 

The Calender Scheduler provides Resources support, with which the single Calender Scheduler is shared by multiple resources simultaneously.

These resources include: (Burundi, UgandaSouth Sudan, Rwanda, Tanzania, Video Conference, International and EAC Headquarters )

 

 

 

Each resource in the Scheduler is arranged in a column/row wise, with individual spacing to display all its respective appointments on a single page. It also supports the grouping of resources, thus enabling the categorization of resources in a hierarchical structure and shows it either in expandable groups (Horizontal view) or else vertical hierarchy one after the other (Vertical view).

One or more resources can be assigned to the Scheduler appointments by making selection of the resource options available in the appointment window.
