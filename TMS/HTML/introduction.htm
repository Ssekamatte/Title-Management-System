Introduction

The EAC Monitoring and Evaluation System (EAMS) is computerized system used to provide M&E team a means to monitor the status of the implementation of decisions and Directives, monitor the Common Market Protocols, Non-Tariff Barriers (NTBs), Protocols, EALA Bills, Single Customs Territory (SCT), Calender of Activities, Reports and Utilities module.

The user manual provides help on how to use the M&E Web Application, designed by the East Africa Community.

Getting started

To access the system, open your web browser from a desktop/laptop computer on your network. In your address bar, type in the web address given to you by your system’s administrator. It is http://eamscentral.eachq.org/Account

This Launches the Login window as shown below. Please note that this is an authenticated system and one needs credentials (User-Name and Password)  to access the system.
